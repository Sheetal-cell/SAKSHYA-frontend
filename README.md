# ⚖️ Sakshya – Frontend (CCMS Legal Intelligence System)

## 📌 Overview
Sakshya Frontend is the UI layer of the CCMS (Judgment Intelligence System).  
It provides a legal AI chat interface, dashboard analytics, and judgment analysis features.

---

## 🚀 Tech Stack
- React 18
- Vite
- React Router DOM
- Axios
- Context API / Hooks

---

## 📂 Project Structure
src/
├── components/
├── pages/
├── assets/

├── main.jsx

## ⚙️ Setup Instructions

### 1. Clone Repository
```bash
git clone https://github.com/your-username/sakshya.git
cd sakshya/frontend

2. Install Dependencies
npm install
3. Create Environment File

Create .env in root:

VITE_API_URL=http://localhost:5000
4. Run Development Server
npm run dev

App runs at:

http://localhost:5173
5. Build for Production
npm run build
